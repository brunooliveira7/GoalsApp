## 🧑🏻‍💻 Tecnologias

Projeto desenvolvido com as seguintes tecnologias:

- JavaScript
- Node.js
- Git e Github

## 💻 Projeto

O Focus Timer divide o trabalho em intervalos curtos e focados, com pausas regulares, otimizando o desempenho e prevenindo a fadiga
